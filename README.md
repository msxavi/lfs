# LFS

## Get Started
 - Clone the repo as non-root user;
 - As root, execute `bash lfs/install-requirements.sh`
 - Check if everything is ok by running: `bash lfs/host-check.sh`
 - Check if everything is ok by running: `bash lfs/host-check.sh`
